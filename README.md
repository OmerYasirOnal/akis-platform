# Basit Sayaç Uygulaması

## Proje Açıklaması
Kullanıcıların bir sayıyı kolayca takip edebildiği, artır ve azalt butonları aracılığıyla sayıyı yönetebileceği basit ve kullanışlı bir sayaç uygulamasıdır.

## Teknoloji Yığını
- React 18
- Vite 5
- CSS3

## Kurulum Adımları

```bash
npm install
npm run dev
```

Tarayıcınızda `http://localhost:5173` adresine gidin.

## Özellikler
- Sayaç başlangıçta 0 değerinde görüntülenir
- Artır butonu ile sayaç bir birim artırılabilir
- Azalt butonu ile sayaç bir birim azaltılabilir
- Arka arkaya tıklamalarla sayaç doğru şekilde güncellenir
- Sıfırla butonu ile sayaç başlangıç değerine döndürülebilir
