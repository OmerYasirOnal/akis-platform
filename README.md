# Basit Sayaç Uygulaması

## Proje Açıklaması
Kullanıcıların bir sayıyı artırıp azaltabileceği, anlık değeri ekranda görebileceği basit ve sezgisel bir sayaç uygulamasıdır. Temel sayma işlemlerini hızlı bir şekilde gerçekleştirmeyi sağlar.

## Teknoloji Yığını
- React 18
- Vite
- CSS3

## Kurulum Adımları

```bash
npm install
npm run dev
```

Tarayıcıda `http://localhost:5173` adresini açın.

## Özellikler
- Sayacı 1 artırma (Artır butonu)
- Sayacı 1 azaltma (Azalt butonu)
- Anlık sayaç değerini ekranda görüntüleme
- Başlangıç değeri: 0
- Sayfa yenilendiğinde sayaç sıfırlanır
