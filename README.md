# Basit Sayaç Uygulaması

## Proje Açıklaması
Kullanıcıların tek bir ekranda sayıyı artırıp azaltabildiği, anlık değeri görebildiği minimal bir sayaç uygulamasıdır. Sayım işlemlerini hızlı ve sezgisel biçimde gerçekleştirmeyi sağlar.

## Kurulum Adımları

```bash
npm install
npm run dev
```

Tarayıcıda `http://localhost:5173` adresini açın.

## Teknoloji Yığını
- React 18
- Vite
- CSS3

## Özellikler
- Sayaç başlangıçta 0 değerinde görüntülenir
- **Artır** butonuna tıklayarak sayacı 1 birim artırabilirsiniz
- **Azalt** butonuna tıklayarak sayacı 1 birim azaltabilirsiniz
- Sayaç değeri, Artır ve Azalt butonları aynı anda ekranda görünür
- Anlık sayaç takibi sağlar
