# Basit Sayaç Uygulaması

## Proje Açıklaması
Kullanıcıların bir sayıyı kolayca artırıp azaltabildiği, tek ekranlı sezgisel bir sayaç uygulamasıdır. Anlık sayı göstergesi ve artır/azalt butonlarıyla hızlı kullanım sağlar.

## Teknoloji Yığını
- React 18
- Vite
- CSS3

## Kurulum Adımları

```bash
npm install
npm run dev
```

Tarayıcınızda `http://localhost:5173` adresini açın.

## Özellikler
- Sayacı bir artırma (Artır butonu)
- Sayacı bir azaltma (Azalt butonu)
- Anlık sayı değerini ekranda görüntüleme
- Başlangıç değeri 0 olarak ayarlanmış
- Ardışık tıklamalarda değer sırasıyla güncellenir (1, 2, 3, ...)
