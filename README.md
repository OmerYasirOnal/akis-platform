# Vanilla JS Basit Sayaç Uygulaması

## Proje Açıklaması
Kayıt veya kurulum gerektirmeden anlık sayım yapmanızı sağlayan sade ve hızlı bir sayaç uygulamasıdır. Artır ve Azalt butonları ile sayacı kolayca yönetebilirsiniz.

## Teknoloji Yığını
- React 18
- Vite 5
- CSS3

## Kurulum Adımları

```bash
npm install
npm run dev
```

Tarayıcınızda `http://localhost:5173` adresini açın.

## Özellikler
- Sayfa yüklendiğinde sayaç 0 olarak başlar
- Artır butonu ile sayacı 1 birim artırabilirsiniz
- Azalt butonu ile sayacı 1 birim azaltabilirsiniz
- Negatif değerler desteklenir ve ekranda gösterilir
- Sayfa yenilendiğinde sayaç sıfırlanır (veri kalıcılığı yoktur)
- Sade ve kullanıcı dostu arayüz
