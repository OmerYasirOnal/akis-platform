## Açıklama

TypeScript ekosisteminde React frontend ve Node.js backend geliştirme hizmeti sunuyorum.

**Neler yapabilirim:**
- React (Vite + Tailwind) ile modern, responsive web uygulaması
- Fastify/Node.js ile REST API geliştirme
- PostgreSQL veritabanı tasarımı ve Drizzle ORM entegrasyonu
- Test yazımı ve kalite kontrolü
- Mevcut projelere yeni özellik ekleme veya hata düzeltme

**Çalışma şeklim:**
1. **Analiz** — İhtiyaçlarınızı anlamak için detaylı mesajlaşma veya kısa görüşme
2. **Geliştirme** — Düzenli ilerleme güncellemeleriyle temiz kodlama
3. **Test** — Teslimattan önce kapsamlı test
4. **Revizyon** — Memnun kalana kadar düzeltme ve iyileştirme

**Kullandığım teknolojiler:**
TypeScript, JavaScript, React, Vite, Tailwind CSS, Node.js, Fastify, PostgreSQL, Drizzle ORM, REST API, Git

**Neden ben:**
- 1.300+ testli projede çalışma tecrübesi (production-grade kalite)
- Açık iletişim ve düzenli güncelleme
- Geri bildirime hızlı yanıt

Projenizi konuşalım! Gereksinimlerinizi mesaj olarak gönderin, size özel teklif hazırlayayım.
