## 2. CV Project Bullets (6 Core)

Use these under "Projects" section of CV. All are real, from AKIS Platform:

1. End-to-end development of a TypeScript platform with Fastify backend and React + Vite SPA architecture
2. Database schema design and data access layer implementation using PostgreSQL + Drizzle ORM with migration-driven development
3. Contract-first agent execution: defined input/output schemas, error envelope standards, and deterministic output validation for 3 autonomous agents
4. Real-time job/agent execution tracking via SSE-based event streaming with observability-focused UI (phase progress, inner monologue, expanding file tree)
5. External system isolation through adapter pattern (MCP concept) — decoupled integrations for GitHub, Jira, and Confluence
6. CI/test discipline: 1,391 tests across backend (842) and frontend (549), small reviewable changes, verification-driven iteration
