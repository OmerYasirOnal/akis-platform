# AKIS Workstream Roadmap

## Phase 0: MVP (Current)
- Profile onboarding
- Manual job ingest
- Deterministic matching with explanation JSON
- Proposal draft generation
- Audit logging

## Phase 1: Payments + Escrow
- Milestone contracts
- Escrow provider integration
- Invoice and payout workflow
- Dispute lifecycle dashboard

## Phase 2: Integration Marketplace
- Connector marketplace for external platforms
- Source-specific ingestion schedulers
- Rate limit and compliance management
- Partner API governance panel

## Phase 3: Multi-language + Regionalization
- Full multilingual UX
- Region-specific policy/compliance packs
- Localized pricing and payment rails
- Regional trust and verification add-ons

## Exit Criteria by Phase
- Phase 0: user can onboard -> ingest -> match -> generate proposal locally.
- Phase 1: escrow-backed milestone lifecycle end-to-end.
- Phase 2: connector onboarding and monitoring self-service.
- Phase 3: region-aware operation with language and compliance support.
