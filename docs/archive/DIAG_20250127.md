## Support & Troubleshooting

### Common Issues

**Q: "Missing credentials in environment"**

A: Add `GITHUB_APP_ID`, `GITHUB_APP_INSTALLATION_ID`, and `GITHUB_APP_PRIVATE_KEY_PEM` to `.env.local`.

**Q: "Token exchange failed: 401"**

A: Verify App ID and private key. Check installation status.

**Q: "Module not found: jsonwebtoken"**

A: Run `npm install jsonwebtoken`.

**Q: "PR creation failed: No commits between main and branch"**

A: Enable `forceCommit: true` option to add timestamp file.
