# AKIS Prompt Pack (Guard-Railed Task Starters)

Use these prompts as **WHAT/WHERE/RULES** task starters. They never dictate implementation details; they bind Cursor to AKIS architecture:
- Modular Monolith
- Backend: Fastify + TypeScript + Drizzle + PostgreSQL
- Frontend: React (Vite) + Tailwind SPA
- External integrations via MCP Client Adapters only
- Agent patterns: Factory/Registry, Orchestrator, FSM, Strategy/Command, plan→execute→reflect (Planner/Reflector), DI from Orchestrator

Each prompt includes:
- TASK (what artifact to create)
- LOCATION (exact file path)
- RULES & CONSTRAINTS (non-negotiables)
- ACCEPTANCE CRITERIA (lint-like checks)
- OUT OF SCOPE (forbidden items)
